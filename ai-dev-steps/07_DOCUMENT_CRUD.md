# 阶段 7：文档新增、编辑、删除

> ✅ 已完成（2026-06-18）

## 本阶段目标

支持基础文档维护，包括新增、编辑、删除。

本阶段不实现导入导出功能。

## 给 AI 的提示词

```text

本次只实现【阶段 7：文档新增、编辑、删除】。

目标：
支持基础文档维护。

要求：
1. 支持新建文档。
2. 支持编辑已有文档。
3. 支持删除文档。
4. 删除前必须有确认框。
5. 新建和编辑时必须校验 JSON。
6. 操作成功后刷新文档列表。
7. 渲染进程通过 IPC 调用主进程。
8. 不要实现导入导出功能。

```

## 需要调用的 ES API

不指定 ID 新增：

```http
POST /{index}/_doc
```

指定 ID 新增或更新：

```http
PUT /{index}/_doc/{id}
```

删除：

```http
DELETE /{index}/_doc/{id}
```

## 推荐 IPC 接口

```text
document:create
document:update
document:delete
```

## 新建文档要求

```text
1. 选择索引。
2. 输入 _id，可选。
3. 输入 JSON 文档。
4. 校验 JSON。
5. 提交成功后刷新列表。
```

## 编辑文档要求

```text
1. 从文档列表打开编辑弹窗。
2. 显示完整 _source JSON。
3. 修改 JSON。
4. 保存后调用更新接口。
5. 成功后刷新列表。
```

## 删除文档要求

```text
1. 删除按钮放在文档列表行操作中。
2. 删除前弹出确认框。
3. 用户确认后执行删除。
4. 成功后刷新列表。
```

## 本阶段验收标准

```text
1. 能新增文档。
2. 能编辑文档。
3. 能删除文档。
4. 删除前有确认。
5. JSON 非法时不能提交。
6. 操作成功后列表刷新。
7. 未实现导入导出功能。
```
