# V0.1.0 发布说明 · 阶段 1～10 汇总

> 版本：V0.1.0  
> 发布日期：2026-06-18  
> 范围：阶段 1 ~ 阶段 10

本版本是 SearchDBPad 的第一个对外可交付版本（V0.1），完成了从项目骨架到 Windows 打包的完整闭环。

## 阶段清单与时间

| 阶段 | 内容 | 完成日期 |
|---|---|---|
| 1 | 项目初始化（Electron + React + TS + Vite + AntD 骨架） | 2026-06-18 |
| 2 | 连接管理（CRUD + 测试连接 + electron-store 持久化） | 2026-06-18 |
| 3 | 集群信息 + 索引列表 | 2026-06-18 |
| 4 | 索引详情（Mapping / Settings） | 2026-06-18 |

| 9 | 导入（JSON / NDJSON / CSV，Bulk API） | 2026-06-18 |
| 10 | 打包和发布（electron-builder，Windows .exe） | 2026-06-18 |

## V0.1.0 交付能力总结

V0.1.0 完成以下闭环：

```text
连接管理 → 集群/索引浏览 → 索引详情 → 文档查询 → 文档 CRUD → 导入导出 → Windows 打包
```

具备的能力：

- 连接管理：保存 / 编辑 / 删除 / 测试 ES 连接（无认证 + Basic Auth）
- 集群信息：集群名称、版本、健康状态、节点数、索引数
- 索引列表：_cat/indices，支持刷新、按名称搜索
- 索引详情：Mapping / Settings 查看
- 文档查询：默认 match_all 分页、DSL 编辑器（Monaco）、简单查询表单
- 文档 CRUD：新增 / 编辑 / 删除，含删除二次确认
- 导入：JSON 数组 / NDJSON / CSV，Bulk API 写入，含成功 / 失败统计
- 导出：JSON / NDJSON / CSV，CSV 含 UTF-8 BOM 兼容 Excel
- 打包：electron-builder，NSIS + Portable
